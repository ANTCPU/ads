## 🧠 INSIGHTS FROM TODAY

- Agent voices (Aria, Herald, Scout, Forge) are the
  personality layer of the notification system.
  Each agent owns a domain:
  · Aria     → ad quality + review
  · Herald   → engagement + referrals + digests
  · Scout    → analytics + performance
  · Forge    → upgrades + billing + tier changes

- Admin as conductor — live user dashboard preview
  is a feature. You see exactly what they see.
  Keep it, protect it with PIN.

- The dotenv injection from vestauth agent was
  intercepting env loads. Always use override:true
  or hardcode values when debugging Supabase directly.

- Build tool: Turbopack — apostrophes in single-quoted JS strings will break builds
